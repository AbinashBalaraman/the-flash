# DailyAI — AI-Native Newsroom

**DailyAI** is a next-generation, AI-native news platform that autonomously discovers, analyzes, and reports on the cultural and technological trends shaping our world. 

By leveraging high-performance large language models (LLMs) via NVIDIA NIMs, DailyAI eliminates traditional editorial bottlenecks to deliver real-time, substantive reporting with a sharp, analytical edge.

## 🚀 Vision
In a world of information overload, DailyAI acts as a high-fidelity filter. It doesn't just aggregate; it interprets. Every article, digest, and insight is generated by a conversational intelligence trained to provide depth, context, and the unexpected "editor's angle."

## 🛠️ Tech Stack
- **Frontend**: High-speed Vite-powered SPA with a custom-built, glassmorphic design system.
- **Backend**: Netlify Serverless Functions for secure, scalable logic.
- **AI Engine**: NVIDIA NIM (Large Language Model Inference) for near-instantaneous content generation.
- **Data Sourcing**: Real-time news integration with multi-source fallback.

## 📦 Features
- **The Feed**: Dynamically generated article cards covering trending topics.
- **Deep Dives**: Full-length, AI-authored articles triggered on demand.
- **Daily Digest**: A structured briefing on the day's essential stories.
- **Ask Signal**: An interactive Q&A layer for every article.
- **Signal Chat**: A persistent conversational assistant for deeper trend exploration.

## ⚙️ Setup & Deployment

### 1. Environment Variables
To run DailyAI, you must provide your NVIDIA API key as an environment variable:

- `NVIDIA_API_KEY`: Your NVIDIA API key (used for LLM inference).
- `NVIDIA_BASE_URL`: (Optional) Defaults to NVIDIA's primary integration endpoint.

### 2. Local Development
```bash
# Install dependencies
npm install

# Start the development server
# Ensure NVIDIA_API_KEY is set in your environment
npm run dev
```

### 3. Netlify Deployment
DailyAI is built for Netlify. Simply link your repository and add the `NVIDIA_API_KEY` to your Netlify site settings under **Environment Variables**.
